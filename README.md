# chirpstack-lorawan-setup  
This repository contains Python scripts and documentation for setting up a ChirpStack LoRaWAN environment. It includes examples for creating tenants, applications, gateways, device profiles, and devices via the ChirpStack REST API.  

## Contents  
- `setup_chirpstack.py` – Python script to automate creation of tenants, applications, gateways, and devices using the ChirpStack REST API.  
- `agents.md` – Documentation about the agent tasks and guidelines used in this project.
